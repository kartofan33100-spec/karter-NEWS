## Описание проекта

Karter NEWS — это веб-портал для публикации новостей и статей, разработанный в рамках курсового проекта.  
Сайт позволяет пользователям просматривать новости, искать статьи по названию, фильтровать публикации по категориям, а зарегистрированным пользователям — создавать, редактировать и удалять собственные статьи.

Проект реализован по клиент-серверной архитектуре и включает frontend, backend, базу данных, систему аутентификации и деплой в облаке.

---

## Цель проекта

Разработать современный веб-портал для публикации новостей и статей с удобным интерфейсом, системой регистрации и авторизации, поиском, категориями и возможностью управления публикациями.

---

## Задачи проекта

- изучить особенности новостных веб-порталов;
- проанализировать существующие решения;
- спроектировать структуру системы;
- разработать клиентскую часть сайта;
- разработать серверную часть и REST API;
- подключить базу данных;
- реализовать регистрацию и вход пользователей;
- реализовать создание, редактирование и удаление статей;
- реализовать поиск и фильтрацию по категориям;
- провести тестирование и развернуть проект.

---

## Основной функционал

### Для всех пользователей
- просмотр главной страницы с новостями;
- просмотр отдельной статьи;
- поиск статьи по названию;
- переход по категориям:
  - Политика
  - Спорт
  - Финансы
  - Здоровье
- страница 404 для неверных маршрутов.

### Для зарегистрированных пользователей
- регистрация;
- вход в аккаунт;
- сохранение авторизации после обновления страницы;
- создание статьи;
- редактирование собственной статьи;
- удаление собственной статьи;
- просмотр страницы Мои статьи;
- защита от редактирования и удаления чужих статей.
